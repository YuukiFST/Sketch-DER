Script do DER
Exemplo — RH

Sistema com funcionário e departamento. 

# DER: Sistema de RH

ENTIDADE funcionario
  id_funcionario   serial       PK
  nome             varchar(100)
  cpf              char(14)
  salario          numeric(7,2)
  data_admissao    date
  email            varchar(100)
FIM

ENTIDADE departamento
  id_departamento  serial       PK
  nome             varchar(50)
  sigla            varchar(5)
  gestor           varchar(50)
FIM

RELACIONAMENTO trabalha
  DE funcionario    (1,1)
  PARA departamento (0,n)
FIM
Exemplo — Pedidos

Sistema com 3 entidades e 2 relacionamentos. 

# DER: Sistema de Pedidos

ENTIDADE cliente
  id_cliente   serial       PK
  nome         varchar(100)
  email        varchar(100)
  telefone     varchar(20)
FIM

ENTIDADE pedido
  id_pedido    serial       PK
  data         date
  status       varchar(20)
  total        numeric(10,2)
FIM

ENTIDADE produto
  id_produto   serial       PK
  nome         varchar(100)
  preco        numeric(8,2)
  estoque      integer
FIM

RELACIONAMENTO realiza
  DE cliente   (1,1)
  PARA pedido  (0,n)
FIM

RELACIONAMENTO contem
  DE pedido    (1,1)
  PARA produto (1,n)
FIM
Palavras-chave
ENTIDADE nome Declara uma entidade
FIM Encerra o bloco
RELACIONAMENTO nome Declara um relacionamento
DE ent (x,x) Entidade origem + cardinalidade
PARA ent (x,x) Entidade destino + cardinalidade
# comentário Linha ignorada
Atributos
ENTIDADE nome
  nome_campo   tipo   [PK]
FIM
PK Chave primária — bolinha azul preenchida, nome sublinhado
serial Auto-incremento
varchar(N) Texto variável
integer / int Inteiro
boolean Verdadeiro/Falso
timestamp Data e hora
text Texto longo
decimal(X,Y) Número decimal
Cardinalidades
(1,1) Exatamente um
(0,1) Zero ou um
(0,n) Zero ou muitos
(1,n) Um ou muitos
Comandos do Mouse

• Mover Tela (Pan) — segure o Botão Direito no fundo do canvas e arraste.
• Seleção Múltipla (Lasso) — segure o Botão Esquerdo no fundo para selecionar vários itens.
• Mover em Bloco — após selecionar, arraste qualquer item do grupo.
• Mover Cardinalidade — arraste individualmente os rótulos de (1,n).
• Zoom — use o scroll do mouse ou os botões no topo do canvas.

Teclas de Atalho

• Ctrl + Z — Desfazer: reverte a última alteração.
• Ctrl + Y — Refazer: avança uma alteração desfeita.
• O — Organizar: selecione uma entidade e aperte O para enfileirar atributos.
• R — Inverter Lado: selecione a entidade e aperte R para espelhar a lista de atributos.
• Ctrl + Enter — Gerar DER: atualiza o diagrama com o código atual.
• Clique — Editar: abre o painel de propriedades do item.
